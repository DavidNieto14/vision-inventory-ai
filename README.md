# Vision Inventory AI – Línea de Pintura

Proyecto académico orientado al diseño de un sistema de visión por computadora para reducir la discrepancia entre inventario físico y digital en una línea de pintura industrial.

## Objetivo

Reducir en al menos 30% la discrepancia actual mediante conteo automático basado en visión por computadora.

## Metodología

El proyecto se desarrolla bajo marco Scrum con sprints de dos semanas.

## Estructura del repositorio

- docs/ → Documentación técnica
- src/ → Código fuente
- data/ → Datos de prueba
